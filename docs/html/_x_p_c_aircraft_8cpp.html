Detailed Description

XPCAircraft represents an aircraft as managed by xplanemp2.

Note
This class bases on and is compatible to the XPCAircraft wrapper class provided with the original libxplanemp. In xplanemp2, however, this class is not a wrapper but the actual means of managing aircraft. Hence, it includes a lot more members.
