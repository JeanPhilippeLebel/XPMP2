dataRefs available to CSL Models

XPMP2 provides the following dataRef values for each CSL model rendered. That means, a CSL model can use it for its animations.

Actual values are taken from the XPMP2::Aircraft::v array. Compare the indexes listed in the second comlumn as defined in the enum DR_VALS (XPMPAircraft.h).
