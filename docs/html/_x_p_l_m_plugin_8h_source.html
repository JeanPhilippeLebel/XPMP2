168 /*
169  * Plugin messages are defined as 32-bit integers. Messages below 0x00FFFFFF
170  * are reserved for X-Plane and the plugin SDK.
171  *
172  * Messages have two conceptual uses: notifications and commands. Commands
173  * are sent from one plugin to another to induce behavior; notifications are
174  * sent from one plugin to all others for informational purposes. It is
175  * important that commands and notifications not have the same values because
176  * this could cause a notification sent by one plugin to accidentally induce a
177  * command in another.
178  *
179  * By convention, plugin-defined notifications should have the high bit set
180  * (e.g. be greater or equal to unsigned 0x8000000) while commands should have
181  * this bit be cleared.
182  *
183  * The following messages are sent to your plugin by x-plane.
184  *
185  */
186 
187
